### Create a complete tag

```bash
git tag -a myTag -m 'My tag comment'
```

### Push this tag on remote host

```bash
git push origin myTag
```

### Delete a remote tag

```bash
git push --delete origin myTag
```

### Delete a local tag

```bash
git tag -d myTag
```
