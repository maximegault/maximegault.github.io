## System commands

### Consoles

* generic one: `mmc`
* users and groups: `lusrmgr.msc`

### Create aliases

* to a database: with `cliconfg`:
  * `Alias` tab
  * select `TCP/IP`
  * type both alias in `Server alias` and instance\database in `Server name`
